## üöÄ How to Deploy on Vercel

Deploying your site to Vercel is incredibly simple and takes less than 2 minutes.

### Option 1: Using the Vercel Website (Recommended)
1. **Login/Sign up** at [vercel.com](https://vercel.com).
2. Click **"Add New"** -> **"Project"**.
3. **Import** your repository `ayushdalvi/volyx-landing-page` from GitHub.
4. **Configure Project:**
   - Framework Preset: `Other` (or it will auto-detect as Static).
   - Root Directory: `./`
5. Click **Deploy**. Vercel will give you a live URL in seconds!

### Option 2: Using the Vercel CLI
If you prefer the terminal, run these commands:
```bash
# Install Vercel CLI globally
npm i -g vercel

# Login to your account
vercel login

# Deploy the project
vercel
```
