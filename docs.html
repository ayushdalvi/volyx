Installation

Installing Volyx is simple. Follow these steps to get started:

1. Download the installer from the home page.
2. Run the .exe file (Windows 10/11 recommended).
3. Follow the on-screen instructions.

Key Features

Split View

Maximize your productivity by viewing two tabs side-by-side. Simply right-click a tab and select "Split View".

AI Integration

Volyx comes with a built-in AI assistant to help you summarize pages, write code, or answer questions without leaving your browser.

Shortcuts

Boost your speed with these essential hotkeys:

  • Ctrl + T - New Tab
  • Ctrl + W - Close Tab
  • Ctrl + Shift + S - Toggle Split View
  • Ctrl + L - Focus Address Bar
